# SnapRoll – AI-Powered Attendance System

SnapRoll is an AI-powered attendance management system that automates attendance using **Face Recognition** and **Voice Verification**. It provides a secure and user-friendly platform for managing students, teachers, and attendance records.

## ✨ Features
- Face Recognition-based Attendance
- Voice Verification
- Teacher Authentication
- Student Management
- Attendance History
- Streamlit Web Interface

## 🛠️ Tech Stack
- Python
- Streamlit
- OpenCV
- face_recognition
- Supabase
- Speech Recognition

## 🚀 Getting Started

```bash
git clone https://github.com/<Vaibhav1o1>/snaproll-ai-attendance-system.git
cd snaproll-ai-attendance-system
pip install -r requirements.txt
streamlit run app.py
```
